All the files in this folder correspond to input files for cluster3.py, which takes in some array of images and returns the filenames
of the three most similar photos in the set. Ideas inspired here:

https://www.pyimagesearch.com/2014/09/15/python-compare-two-images/
- This document touches on mean squared errors method and skimage's structural image index, both of which were used in this implementation
- In addition, some sample code provided on this website was taken for the development of cluster3.py

Online run-time environment:
